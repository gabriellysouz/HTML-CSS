Usando links externos

É possivel colocar links que direcionam para outros sites (externo) usando a tag ancora: <a>. É recomendavel utilizar a target _blank (<a href="https://github.com/gustavoguanabara" target="_blank" rel="external>) para que o site linkado nao sobreponha o seu. Segue exemplo:

Voce pode acessar meu repositorio publico no GitHub

Voce tambem pode acessar o meu canal gratuito no YouTube

Usando links internos

Link interno faz relação com outra pagina que esta no nosso servidor que diz respeito ao nosso proprio site. Para isso, a proxima pagina precisa estar na mesma pasta da pagina principal.

Nesse exemplo para acessar a segunda pagina, foi criado um segundo arquivo com o nome "pag002" e feita a seguinte ancora: <a href="pag002.html" rel="next">

Ja para a pagina de noticias, foi criado uma nova pasta chamada "noticias" e dentro dessa pasta, o arquivo "pag003". Foi usado entao o caminho <a href="noticias/pag003.html" rel="next">

Esta é a primeira pagina do site. Se voce quiser, pode acessar tambem a minha segunda pagina

Voce tambem pode acessar a nossa pagina de noticias

Como a pagina "pag003" se encontra em outra pasta, usamos o caminho <a href="../index.html"> para retornar a pagina "index" (principal).
