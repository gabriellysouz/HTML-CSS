Videos com HTML5

Videos hospedados internamente

Assim como com as imagens, para colocar videos no site, que estao hospedados internamente, existem duas maneiras:
A 1º é utilizando a tag <video> com os parametros src='' para o endereço do arquivo, width para definir o tamanho na tela e controls para colocar o play no video.

Este video esta hospedado no meu proprio servidor utilizando o codigo <video src="midia/video-gatinho.mp4" width="200" controls>


Na 2º maniera, assim como com as imagens, abrange mais formatos, facilitando a experiencia do usuario e aumentando a compatilibilidade com o mesmo. Atenção na ordem dos sources. O codigo é lido "se o primeiro nao der certo, tenta o segundo... e assim sucessivamente". Nesse caso, o padrao (mais leve) sempre é o primeiro valor.

Videos de hospedagem externa

É possivel colocar videos no site que estao hospedados em outro servidor, como por exemplo o YouTube e Vimeo.

Para adicionar é facil, utilizamos a opção 'incorporar' na opção de compartilhar do YouTube ou Vimeo. Depois é só copiar o codigo gerado e colar no nosso documento HTML.
