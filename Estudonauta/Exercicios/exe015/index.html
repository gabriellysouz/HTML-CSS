Estilos Externos

Estilos externos consiste em abrigar toda formataçao CSS num arquivo separado e usar o link:css na pagina para que assuma os estilos. No exemplo abaixo, foi criado o arquivo 'style' para abrigar a formataçao e nas duas paginas foi linkado esse style.
É importando ressaltar que é possivel utilizar as 3 maneiras dentro do mesmo arquivo. O exemplo abaixo utiliza essas 3 tecnicas.
* usar inline apenas pontualmente, CSS interno só Utilizar para estilos pequenos, o CSS externo é o mais versatil e recomendado principalmente se o site tiver mais paginas
