Listas ordenadas

Listas ordenadas sao as que a ordem da lista importa, por exemplo,passo-a-passo, listas de carros mais caros etc. Para isso usamos a tag <ol> e cada item é identificado pela tag <li>.

É possivel personalizar o tipo de marcador usando o parametro type: <ol type=""> e personalizar o inicio usando o parametro start.

Lista de definiñçoes

É possivel criar lista de definição usando <dt> para definir o termo e <dd> para descrever. Exemplo:

    Termo - HTML
    Descriçao - Linguagem de marcação para criaçao do conteudo de um site
    CSS
    Liguagem de marcação para criação do design de um site
    JavaScript
    Linguagem de programação para criação de interatividade de um site
